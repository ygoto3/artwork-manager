# artwork-manager

A web service to manage artworks (in development)

## Requirement

- Node.js v8.1.3~
- MongoDB v3.4.6~

## Run

```sh
$ yarn build
```

Then

```sh
$ yarn start
```

## Development

### Server-side

```sh
$ yarn dev:server
```

Then

```sh
$ yarn start:dev
```

### Client-side

```sh
$ yarn dev:client
```

### UI Components

```sh
$ yarn storybook
```

Then

```sh
$ yarn dev:client
```

### Test

```sh
$ yarn test
```
